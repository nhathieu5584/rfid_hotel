# rfid_hotel
B1: Tải Xampp -> Trong Xampp Control Panel: Bật Apache và MySQL
    Sau đó tải folder rfid_hotel về và copy folder đấy vào đường dẫn C:\xampp\htdocs
B2: Truy cập: http://localhost/phpmyadmin/
B3: Tạo 1 database tên là rfid_hotel
B4: sau đó Import file rfid_hotel.sql vào
B5: Giờ đã có thể truy cập vào website
Địa chỉ: http://localhost/rfid_hotel/
Tên miền của website cũng chính là tên của folder chứa source
